# Admin – Manage Employees (Phase 1)

## Purpose
Allow admins to view and manage employees.

## Route
/(admin)/employees

## MVP Features
- View list of employees
- See role and status

## Deferred
- Creating employees
- Editing employees
- Invitations

## Notes
UI-first approach. Supabase integration will follow after routing is stable.

## Phase 1A – Employee List
- Employees loaded from Supabase
- Read-only access
- Admin only
