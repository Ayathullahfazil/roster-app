# Project Overview

This application is an employee roster and workforce management system.

The goal of the project is to deliver a usable MVP quickly with core features,
while allowing additional features to be added incrementally without redesign.

## Tech Stack

- Expo (React Native + Web)
- Supabase (Auth + Database)
- Email authentication (initially)
- Role-based access (Admin, Employee)
